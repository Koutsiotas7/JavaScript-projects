# Image Slider Program

## Description
An interactive image slider built with HTML, CSS, and JavaScript that allows users to navigate through a series of images.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Next and previous navigation buttons
- Smooth transition between images
- Simple and responsive UI
