# Calculator Program

## Description
A simple calculator application built with HTML, CSS, and JavaScript that performs basic arithmetic operations through an interactive user interface.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Addition, subtraction, multiplication, and division
- Clear/reset functionality
- Button-based user input
- Responsive and simple UI
