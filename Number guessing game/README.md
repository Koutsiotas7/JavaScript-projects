# Number Guessing Game

## Description
A simple interactive number guessing game built with HTML, CSS, and JavaScript where the user attempts to guess a randomly generated number.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Random number generation
- User input for guesses
- Feedback on guesses (too high/too low)
- Restart game functionality
