# Rock Paper Scissors Program

## Description
A classic Rock‑Paper‑Scissors game built with HTML, CSS, and JavaScript that lets the user play against the computer.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- User vs Computer gameplay
- Random computer choice each round
- Score tracking
- Simple and responsive UI
