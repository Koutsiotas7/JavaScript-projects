# My Weather App Program

## Description
A front-end JavaScript weather application using HTML, CSS, and JavaScript that fetches and displays weather information based on user input.

## How to Run
1. Clone the repository
2. Open `index.html` in your browser

## Features
- Search weather by city
- Displays temperature and conditions
- Interactive UI
