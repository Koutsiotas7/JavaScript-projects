# Counter Program

## Description
A simple interactive counter application built with HTML, CSS, and JavaScript that allows the user to increase, decrease, and reset a numeric value.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Increase counter
- Decrease counter
- Reset counter
- Real-time UI update
