# Dice Roller Program

## Description
A simple interactive dice roller built with HTML, CSS, and JavaScript that simulates rolling a six-sided dice and displays the result.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Generates a random number from 1 to 6
- Displays the result visually
- Simple interactive UI
